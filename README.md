# Comfy Sloth

An ecommerce application built using HTML,CSS,JavaScript,
ReactJS,ContextAPI,React Router,Stripe(for payments) & Auth0 for 
authentication purpose.

## [Demo Link](https://comfy-app.netlify.app/)
